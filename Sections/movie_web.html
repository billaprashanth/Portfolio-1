Movie Search App – HTML, CSS, JavaScript

Project Details

This is a responsive movie searching application developed using HTML, CSS, and JavaScript. Users can search for any movie using a dynamic input bar, and results are fetched from a movie API (like OMDb API). The app displays movie titles, posters, ratings, and descriptions interactively.

Features

  • Live movie search using OMDb or similar API
  • Search bar with input validation
  • Dynamic rendering of movie posters and info
  • Responsive UI with Flexbox and Media Queries
  • Modern, minimal UI with user-friendly layout
