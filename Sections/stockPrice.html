Stock Price Prediction

Project Details

This project predicts the next-day closing price of stocks using historical data. It includes Exploratory Data Analysis, feature engineering, and predictive models including LSTM and Linear Regression. The UI is presented using Streamlit.

Features

  • Interactive stock prediction app
  • LSTM and Linear Regression comparison
  • Live stock ticker input and trend display
  • Visualization using Matplotlib and Seaborn
  • Deployable on Streamlit Cloud
