Image Super Resolution using GAN

Project Details

This project uses Generative Adversarial Networks (MAD-GAN and SAGAN) for converting low-resolution images into high-resolution images. The model is trained on image datasets using Python and TensorFlow.

Features

  • MAD-GAN architecture for stable training
  • SAGAN for attention-based enhancement
  • PSNR and SSIM metrics for evaluation
  • Dataset preprocessing with OpenCV
  • Side-by-side result comparison
