Features

  • Add new books to the library collection
  • View all available books
  • Borrow books (availability status updated)
  • Return books to mark them available again
  • Interactive menu-driven command-line interface

How It Works

The project contains a single Python script (LMS.py) that handles:

  • Storage of books and their availability (in memory during session)
  • User prompt to select options like add, borrow, return, view, or exit
  • Input-driven actions to update internal data structures

To run the program, use: python LMS.py in the terminal.
